Apache Solr Ref Guide Proof of Concept

As part of the proposal to move the Ref Guide from Confluence, this site provides demos of a small section of the Solr Reference Guide converted to Asciidoc format and published with various options.

The source for these examples is available in the GitHub repo refguide-asciidoc-poc. See the README in particular for information about the methodologies used to produce these examples.

  • Example using Jekyll: Uses Jekyll to generate HTML pages, including a comments section on each page, and a sidebar navigation.
  • Example using JBake: Uses JBake to generate HTML pages, including a comments section on each page. This is available for comparison with Jekyll-generated pages.
  • Sample PDF: Uses Asciidoctor tools to create a PDF. The PDF is created from the same set of pages used to generate the Jekyll HTML pages for direct comparison.

NOTE: Each example above is a demonstration of capabilities, not a final product ready for publication. Further effort is required before it is ready to replace Confluence.
